# Memory-Enhancer

## 一、Skill概述
- 名称：memory-enhancer
- 版本：v1.0
- 目标：为Agent提供智能记忆管理能力
- 核心价值：重要性评分 + 遗忘机制 + 意图检索

## 二、核心功能

### 2.1 记忆存储
- 重要性评分（LLM自动评估1-10分）
- 记忆分类（事实/偏好/上下文/工具调用）
- 生命周期管理（TTL + 评分淘汰）

### 2.2 记忆检索
- 意图识别（查询类型路由）
- 混合检索（语义+关键词+时间权重）
- 冲突解决（时间戳优先/显式覆盖）

### 2.3 记忆遗忘
- 评分低于阈值自动归档
- TTL过期自动清理
- 用户手动清理接口

## 三、使用方式

### 存储记忆
```
memory_store(content, importance_score, category, ttl)
```

### 检索记忆
```
memory_retrieve(query, intent, limit)
```

### 清理记忆
```
memory_clean(threshold)
```

## 四、实现方案

### 4.1 架构设计
```
Skill策略层（重要性评估、记忆分类）
    ↓
扣子能力层（变量、数据库、长期记忆）
    ↓
检索增强（意图识别、混合检索）
```

### 4.2 数据结构
- 变量：存储核心用户画像
- 数据库：存储结构化记忆表
- 长期记忆：存储对话摘要

## 五、验收标准

### 功能验收
- ✅ 记忆存取、覆盖、清理
- ✅ 跨会话延续
- ✅ 重要性自动评估

### 性能验收
- 注入延迟 <200ms
- 检索延迟 <300ms
- Token节省 >50%

### 质量验收
- 提取准确率 >85%
- 偏好识别准确率 >90%
